## How to Apply These Licenses to a Fork

1. Replace all content in `src/data/` with your own.
2. Replace `public/images/` and `public/resume.pdf`.
3. Update `LICENSE` if you want a different code license.
4. Update this document.
5. Keep attribution to the original code author in a `NOTICE` file if you redistribute the code substantially unchanged.
